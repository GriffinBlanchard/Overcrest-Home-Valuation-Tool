# Overcrest Realty - Home Valuation Tool

## Quick Deploy to Vercel

### Method 1: GitHub (Recommended)
1. Create a GitHub account if you don't have one
2. Create a new repository
3. Upload all these files to the repository
4. In Vercel, click "Add New..." → "Project"
5. Select "Import Git Repository"
6. Choose your repository
7. Click "Deploy"

### Method 2: Direct Upload
1. Zip this entire folder
2. In Vercel, click "Add New..." → "Project"
3. Click "Browse" or drag and drop your zip file
4. Click "Deploy"

### After Deployment
- Vercel will give you a URL like: `https://your-app.vercel.app`
- You can connect a custom domain in Vercel settings
- Share the link with your clients!
